# 								Docker

- Docker是一个工具，能够帮助我们更方便的创建，运行，部署软件。

- Docker允许我们将一个软件和其依赖打包成一个单独的库，更有利于移植可运行的软件。

**虚拟机缺点：**

- 占用资源多
- 运行步骤冗余
- 运行速度慢

**Docker的用途**

- 提供一次性的环境
- 提供弹性云服务
- 组建微服务构架
- 让部署方便了很多

------

1. Docker image是Docker container（一个软件） 的设计蓝图
2. Docker image 包含Dockerfile，依赖和程序大代码
3. 文件Dockerfile中包含一系列的指令用来创建Docker image。
